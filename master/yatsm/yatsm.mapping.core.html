yatsm.mapping.core.result_map(out, results, table, attr_columns, attr_funcs=None, **query_kwds)[source]¶

Parameters:
  • out¶ – 2D or 3D (nband x nrow x ncol) array to fill result data into
  • results¶ – A list of HDF5ResultsStore files
  • table¶ – The table to retrieve the data from
  • attr_columns¶ – Attributes from results table to map. The number of attributes given should be the same as the number of bands in out.
  • attr_funcs¶ – Optionally, provide a function to apply to each attribute described in attr_columns. Please supply None, or an iterable of either a callable object or None for each attribute in attr_columns.
  • query_kwds¶ – Additional search terms to pass to HDF5ResultsStore.query()
Returns:

out, but with desired result file attributes mapped into

the image
