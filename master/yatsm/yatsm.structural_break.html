yatsm.structural_break.cusum_OLS(X, y, alpha=0.05)[source]¶

OLS-CUSUM test for structural breaks

Tested against R’s strucchange package and is faster than the equivalent function in the statsmodels Python package when Numba is installed.

The OLS-CUSUM test statistic, based on a single OLS regression, is defined as:

\[W_n^0(t) = \frac{1}{\hat{\sigma}\sqrt{n}} \sum_{i=1}^{n}{\hat{\mu_i}}\]
Parameters:
  • X¶ – 2D (n_obs x n_features) design matrix
  • y¶ – 1D (n_obs) indepdent variable
  • alpha¶ – Test threshold (either 0.01, 0.05, or 0.10) from Ploberger and Krämer (1992)
Returns:

A named tuple include the the test name, change point (index of y), the test score and pvalue, and a boolean testing if the CUSUM score is significant at the given alpha
Return type:

yatsm.structural_break.cusum_recursive(X, y, alpha=0.05)[source]¶

Rec-CUSUM test for structural breaks

Tested against R’s strucchange package.

The REC-CUSUM test, based on the recursive residuals, is defined as:

\[W_n(t) = \frac{1}{\tilde{\sigma}\sqrt{n}} \sum_{i=k+1}^{k+(n-k)}{\tilde{\mu_i}}\]

Critical values for this test statistic are taken from:

A. Zeileis. p values and alternative boundaries for CUSUM tests.
    Working Paper 78, SFB "Adaptive Information Systems and Modelling
    in Economics and Management Science", December 2000b.
Parameters:
  • X¶ – 2D (n_obs x n_features) design matrix
  • y¶ – 1D (n_obs) indepdent variable
  • alpha¶ – Test threshold
Returns:

A named tuple include the the test name, change point (index of y), the test score and pvalue, and a boolean testing if the CUSUM score is significant at the given alpha
Return type:

Exponentially Weighted Moving Average test

Parameters:
  • y¶ – Time series to test. Should be sorted chronologically
  • lambda_¶ – “Memory” parameter, bound [0, 1]
  • crit¶ – Critical threshold for boundary, given as a scalar multiplier of the standard deviation
  • center¶ – Center time series before calculating EWMA
  • std_type¶ –

    Method for calculating process standard deviation. Calculated using:

    • MR for an estimate based on the “moving range” of the scaled mean
    • SD for the sample standard deviation
    • MAD for the Median Absolute Deviation estimate of standard deviation
Returns:

A named tuple include the the test name, change point (index of y), the test score, and a boolean testing if the EWMA score is significant at the given crit
