yatsm.classification.roi module¶

Utilities for extracting training data from region of interests (ROI)

yatsm.classification.roi.extract_roi(raster, vector, feature_prop=None, all_touched=False)[source]¶

Yield pixel data from src for ROIs in features

Parameters:
  • raster¶ – The rasterio dataset used to extract training data values from
  • vector¶ – A list of features from a polygon vector file as GeoJSON-like
  • feature_prop¶ – The name of the attribute from features containing the ROI labels
  • all_touched¶ – Rasterization option that decides if all pixels touching the ROI should be included, or just pixels from within the ROI
Returns:

A tuple

containing an array of ROI data from src (band x n), the ROI data label (n), and the X and Y coordinates of each data point (n and n sized)
Return type:

tuple (np.ndarray, np.ndarray, np.ndarray, np.ndarray)
