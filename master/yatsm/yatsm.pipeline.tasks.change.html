yatsm.pipeline.tasks.change module¶

Functional wrappers around change detection algorithms

yatsm.pipeline.tasks.change.pixel_CCDCesque(pipe, require, output, config=None)[source]¶

Run yatsm.algorithms.CCDCesque on a pixel

Users should pass to require both X and Y arguments, which are interpreted as:

X, Y = require[0], require[1:]
Parameters:
  • pipe¶ – Piped data to operate on
  • require¶ – Labels for the requirements of this calculation
  • output¶ – Label for the result of this calculation
  • config¶ – Configuration to pass to CCDCesque. Should contain init section
Returns:

Piped output
