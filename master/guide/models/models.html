Science Models¶

The time series model run using the YATSM package command line interface (i.e., yatsm_line and yatsm_pixel) is specified in the configuration file using the YATSM['algorithm'] key:

YATSM:
    algorithm: "AN_ALGORITHM"
    ...

AN_ALGORITHM:
    init: ...
    fit: ...

The value specified for the algorithm key within the YATSM section must be the name of another section of the configuration file that can be used to identify, initialize, and run a time series algorithm included in the yatsm.algorithms module.
